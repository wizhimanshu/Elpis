📌 A clean, modern, animated SaaS interface built with React + SCSS — helping users manage interviews with clarity, focus & confidence.
Soft-glass UI, neon glow gradients, floating blobs & motion-smooth scroll animation bring a premium Apple-like UI experience.

🚀 Getting Started

Follow these steps to clone & run Elpis locally.

git clone https://github.com/your-username/elpis.git
cd elpis
npm install
npm run dev
